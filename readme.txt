SIM Bank настраивается следующим способом:
"COM  порт сим банка"="№ модема":"Слот симки по умолчанию, например 1(позже будет менятся, сохранясь)",...
Пример есть сим банк с COM портом "COM2", и в него подключены модемы с номерами в програме 1,2,3,4,5,6,7,8
тогда строка настройки должна выглядеть так:
COM2=1-1,2-1,3-1,4-1,5-1,6-1,7-1,8-1
